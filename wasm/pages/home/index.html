Welcome to the Playground

This playground, inspired by textfsm.nornir.tech, lets you experiment with various tools and parsers directly in your browser. It's built with Go compiled to WebAssembly (WASM), which means everything runs on the client-side.

Supported Features

  • TextFSM Parsing
  • JSON Schema Validation (coming soon)

Have a feature request? Raise an issue on the GitHub repository.
